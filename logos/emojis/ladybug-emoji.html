Emoji CSS

Want to include ladybug emojis in your HTML? Just include the (minified) stylesheet below, then add <i> tags to insert emoji. That's it!
âœˆ Click the emoji code and it will be copied to your clipboard. 

<link href="https://github.com/ladybug-tools/artwork/raw/master/logos/emojis/ladybug-emoji.css" rel="stylesheet">

<i class="em em-some-emoji"></i>
By default, the emoji will be served as PNGs. To use SVG instead, use the em-svg class instead of em. This will scale better, but has larger file size.
